Heritcraft is a full-stack e-commerce web application designed to showcase and sell handcrafted and heritage-inspired products online. The platform provides users with a seamless shopping experience including product browsing, category-based filtering, secure authentication, and responsive UI design.

The project is built using:

Frontend: React.js Backend: Spring Boot Database: MySQL REST APIs for communication between frontend and backend

Features

User authentication and authorization Product management Category-wise product listing Responsive modern UI Add to cart functionality Backend REST API integration Database connectivity with MySQL

Purpose

The goal of Heritcraft is to support traditional artisans and promote handcrafted products through a modern digital marketplace.
